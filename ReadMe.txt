**โปรดระวังหากมี database ที่ชื่อ project อยู่แล้วในเครื่อง ข้อมูลจะถูกลบ
***เว็บไซต์จะทำงานได้ดีที่สุดในหน้าจอขนาดประมาณ 1600x900 pixel

database และ website จำเป็นต้องใช้งานผ่าน XAMPP โดย start Apache และ MySQL
1. import file "SQL.sql" ลงใน phpMyAdmin ไฟล์จะสร้าง database ที่ชื่อว่า project
2. ย้ายโฟลเดอร์ Website ลงใน 'C:\xampp\htdocs\...'
3. เริ่มต้นใช้งาน โดยเข้าใช้งานผ่าน URL 'localhost\Website\Start.php' ใน browser ของท่าน
